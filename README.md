# Superstore Sales & Profit Performance Dashboard

## Overview
This project analyzes retail sales data using Tableau to uncover key insights across regions, categories, and products.

## Tools Used
- Tableau
- Excel

## Key Features
- KPI cards for Sales, Profit, and Quantity
- Sales and Profit trends over time
- Category and Sub-category analysis
- Interactive filters

## Insights
- Technology category generates highest profit
- Some sub-categories show losses despite high sales
- Regional performance varies significantly
